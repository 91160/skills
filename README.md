# Skills

健康160 AI 编码技能仓库，包含多种可复用的 AI 编码工作流和规范。

## 技能列表

| 技能名称                | 说明                                                                                     |
| ----------------------- | ---------------------------------------------------------------------------------------- |
| front-project-context   | 用于扫描前端项目，提取编码规范、基础配置（常量、请求封装、公共方法等）                   |
| prd-audit               | PRD 需求文档审计                                                                         |

## 下载方式

### 安装单个技能

```bash
npx skills add https://github.com/91160/skills --skill <skill-name>
```

示例：

```bash
# 安装 prd-audit
npx skills add https://github.com/91160/skills --skill prd-audit

# 安装 java-project-creator
npx skills add https://github.com/91160/skills --skill front-project-context 
```
